## 制度變革摘要

**113 年以前（舊制，6 科）**
1. SUBJ01 生物化學與營養學
2. SUBJ02 公共衛生營養學
3. SUBJ03 膳食療養學
4. SUBJ04 團體膳食設計與管理
5. SUBJ05 食品衛生與安全
6. SUBJ06 食品學

**114 年以後（新制，6 科）**
1. SUBJ07 生理學與生物化學（取代 SUBJ01 的生化部分）
2. SUBJ08 營養學（取代 SUBJ01 的營養學部分）
3. SUBJ02 公共衛生營養學
4. SUBJ03 膳食療養學
5. SUBJ04 團體膳食設計與管理
6. SUBJ05 食品衛生與安全

科目題數（115-1 實際狀況）：SUBJ02/05/07/08 各 50 題、SUBJ03/04 各 40 題。
舊制 SUBJ01/06 題數以歷年 PDF 實際為準。

## RAG / 檢索建議

若要檢索特定主題，建議以 `subject_code + primary_topic` 組合做為 metadata
filter，而非只依科目名稱比對字串——因為新舊制命名差異會造成漏檢。
