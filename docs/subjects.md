# 營養師國考六科說明

| 科目代碼 | 科目名稱 | 題數 | 佔分 |
|---------|---------|------|------|
| nutrition | 營養學 | 80題 | 包括巨量/微量營養素 |
| physiology_biochemistry | 生理學與生物化學 | 80題 | 代謝、激素、酵素 |
| dietotherapy | 膳食療養學 | 80題 | 各疾病飲食治療 |
| public_health_nutrition | 公共衛生營養學 | 80題 | 流行病學、介入 |
| food_safety | 食品衛生與安全 | 80題 | 法規、污染、標準 |
| foodservice_management | 團體膳食設計與管理 | 80題 | 供餐、菜單設計 |

## 及格標準
- 總成績平均滿 60 分
- 膳食療養學不得低於 50 分
- 任一科不得零分
